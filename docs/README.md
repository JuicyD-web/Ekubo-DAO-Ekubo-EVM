# Documentation Overview

This directory contains documentation for the Ekubo Protocol EVM implementation.

---

## Contents

- **User Guides:**  
  Instructions for using the protocol and interacting with smart contracts.

- **Developer Guides:**  
  Setup, integration, and contribution instructions for developers.

- **Compliance & Governance:**  
  Details on license compliance, revenue sharing, governance, and reporting.

- **Legal & Financial:**  
  Terms of service, privacy policy, regulatory notes, and financial documentation.

- **Templates & References:**  
  Reporting templates, calculation methodologies, and reference materials.

---

## How to Use

- Browse subfolders for specific documentation topics.
- Follow guides and templates for contributions, compliance, and reporting.
- Refer to legal and financial docs for obligations and requirements.

# Financial Documentation Overview

This directory contains financial documents and records related to the Ekubo Protocol EVM implementation.

---

## Contents

- **Revenue Reports:**  
  Detailed quarterly and annual revenue reports, including breakdowns and payment proofs.

- **Payment Records:**  
  Documentation of all payments made to Ekubo DAO, including transaction hashes and receipts.

- **Revenue Calculations:**  
  Methodologies, scripts, and raw data used to calculate protocol revenue and Ekubo DAO shares.

- **Financial Statements:**  
  Summaries of protocol earnings, expenses, and distributions (if applicable).

- **Audit Records:**  
  Logs and results from financial audits and compliance reviews.

---

## How to Use

- Browse subfolders for specific financial documents.
- Use provided templates for new reports and records.
- Ensure all financial activities comply with the Ekubo DAO Shared Revenue License 1.0 and reporting requirements.

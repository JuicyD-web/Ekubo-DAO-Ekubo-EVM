# Security Audits Overview

This directory contains security audit reports and related documentation for the Ekubo Protocol EVM implementation.

---

## Contents

- **Audit Reports:**  
  Detailed findings from internal and external security audits.

- **Audit Logs:**  
  Records of audit activities, dates, and outcomes.

- **Remediation Notes:**  
  Documentation of fixes and improvements made in response to audit findings.

---

## How to Use

- Review audit reports before deploying or upgrading contracts.
- Reference remediation notes to verify that identified issues have been addressed.
- Use audit logs for compliance and governance reviews.

# SECURITY.md

## Security Policy Overview

Ekubo Protocol EVM prioritizes security in all aspects of development and operation.  
This document summarizes our supported versions, vulnerability reporting process, incident response procedures, best practices, responsible disclosure guidelines, and contact information.

---

## Supported Versions

| Version | Status         | Notes                  |
|---------|---------------|------------------------|
| 1.0.x   | Supported     | Active security updates|
| < 1.0   | Not supported | Beta, use at own risk  |

## Security Best Practices

- Use reentrancy protection, access control, input validation, and safe external calls
- Maintain high code coverage and use static analysis tools
- Conduct regular audits and reviews

For audit information, see [audits/README.md](./audits/README.md).

---

## Responsible Disclosure

- Please allow reasonable time for us to resolve issues before public disclosure
- We will acknowledge and credit valid reports
